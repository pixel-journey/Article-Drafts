# Article Folder Template

**Copy this structure for every new article folder `NN_Slug-Title`**

## Recommended Files to Create

### 1. full-article-draft.md
```markdown
---
 title: "[Catchy Title Here]"
 date: YYYY-MM-DD
 tags: [PixelJourney, Web3, Education, PXJ, PxPortals]
 excerpt: "Short engaging summary for previews and SEO."
---

# [Full Catchy Title]

**Hook / Introduction paragraph** (1-2 paras, emoji, relatable or intriguing)

## Section 1: [Key Aspect 1]

Content...

## Section 2: ...

**Conclusion with strong CTA**

*References / Sources: links to docs pages*

---

**Word count target:** 800-2500 words depending on depth.
```

### 2. x-initial-announcement-post.md

**Primary X Post / Thread Starter**

```
🚀 [Hook emoji + bold statement or question]

[1-2 sentence summary of what the article reveals or why it matters to the community]

Full article 👉 [link or "in bio / comments" or thread below]

#PixelJourney #PXJ #Web3Education [other relevant hashtags]

[Image or carousel suggestion]
```

**If thread:** Add 2-5 more tweets with key points, quotes, fun facts, ending with CTA to read full + engage (reply, RT, bookmark).

### 3. recurring-x-posts.md

**Collection of reusable / evergreen posts for future scheduling or manual posting**

#### Fun Facts (5-10 ideas)
- "Did you know? The Pixal PFPs were the first major drop that kicked off the entire Pixel Journey economy..."
- etc.

#### Throwbacks
- "Throwback to when we first bridged assets to Polygon... The multi-chain vision is becoming reality!"

#### Anniversary / Celebration
- "[X months/years] ago today: [Milestone]! Here's how far we've come together..."

#### Different Angles for Recurring Promotion
- Educational snippet
- Motivational / community love
- Stats or milestone update (when new data available)
- Meme or light-hearted take
- Question to community to spark replies

**Tip:** Vary the copy, never copy-paste the same post. Use these as starting points and refresh with current context.

### 4. x-thread-version.md

Full optimized X thread (or key excerpts + hooks). Use short lines, emojis, line breaks, questions/polls, numbered points if applicable. End with link or "read the full article in the thread / docs / Medium".

### 5. image-graphics-prompts.md

Include subtle PXJ branding or journey elements."

**Carousel / Infographic Prompts** (3-6 slides ideas)
1. ...

**Meme / Fun Visual Prompts**
...

### 6. notes-and-sources.md

- Original docs page(s): [links]
- Key facts / data points to verify
- Team quotes or vision statements
- Community feedback or common questions this article addresses
- Update log: [dates and changes]
- SEO keywords / search intent

## Optional Additions
- `medium-frontmatter-ready.md`
- `discord-post.md`
- `carousel-text.md`
- `email-excerpt.md`
- `social-calendar.md` (suggested dates/times/types for this content)
- `additional-formats/` folder for DOCX/PDF exports (use pdf/docx skills when ready)

## Workflow Reminder
1. Research & outline from BRAINSTORM and source docs
2. Write full draft
3. Adapt to X formats (keep tone consistent: fun, educational, pixel vibe)
4. Create image prompts (then generate visuals separately)
5. Review for accuracy, engagement, CTAs
6. Post / schedule
7. Log updates in notes-and-sources.md

This template ensures every article has maximum reusability and impact across channels!
