All links to the Pixel Journey documentation should point to:

---

## 2. Content Audit Checklist for Early Articles

For each of the first ~40 articles, check the following:

### A. Factual Accuracy & Alignment
- Does the content match the current state of the Px ecosystem (as shown on X @PxJourney and in the live portals)?
- Are any claims outdated (old chain support, old feature descriptions, old timelines)?
- Is there any probable misinformation that may have crept in during earlier drafting?
- Does it align with the official information in https://docs.pixeljourney.xyz ?

### B. Branding & Terminology (Px vs Pixals)
- Is "Px" used correctly as the prefix (PxJourney, PxStaking, PxSwap, PxLearn, PxLanding, etc.)?
- Is "Pixals" used **only** for the PFP collection and never for the overall journey or products?
- No standalone "Pix" branding.

### C. Development Status Language
- Are all Px portals and features described in the **present tense as complete**?
- Remove any "coming soon", "in development", "beta", "planned", or "will launch" language.

### D. Consistency with Official Docs
- Does the article contradict or drift from the information in the docs?
- Are key concepts explained at a similar depth and tone?

### E. Cross-Linking Quality
- Are there natural, helpful links to relevant sections of https://docs.pixeljourney.xyz ?
- Is there a clean "Further Reading & Expanded Topics in the Docs" section?
- Are links specific and accurate (not generic homepage links)?

---

## 3. How to Add Links to the Docs

### Recommended Approach
- Link contextually inside the article where the topic is discussed (not just at the bottom).
- Add a dedicated **Further Reading** section near the end for deeper dives.
- Use descriptive link text (e.g. "Learn more about PxStaking in the official docs").

### Example "Further Reading" Block

```markdown
## Further Reading & Expanded Topics in the Docs

For more in-depth explanations and the latest details, see the official Pixel Journey documentation:

- [PxStaking Overview](https://docs.pixeljourney.xyz/ecosystem/px-staking)
- [How PxSwap Works](https://docs.pixeljourney.xyz/ecosystem/px-swap)
- [Understanding $PXJ Tokenomics](https://docs.pixeljourney.xyz/tokenomics/pxj)
- [Getting Started Guide](https://docs.pixeljourney.xyz/getting-started)
```

## 4. Common Issues Expected in the First ~40 Articles

- References to features that have since changed or expanded
- Overly cautious "in development" language that should now be confident and complete
- Missing or weak links to the official docs
- Terminology drift between articles and https://docs.pixeljourney.xyz
